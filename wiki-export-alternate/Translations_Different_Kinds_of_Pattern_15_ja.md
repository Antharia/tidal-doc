``` Haskell
[ ]
```

の代わりに

``` Haskell
{ }
```

を使うと、別の効果が得られます。

``` Haskell
[ ]
```

は、シーケンスの両半分がサイクルにフィットします(ポリリズム)。

``` Haskell
{ }
```

は、パルスは左側のパターンで設定されます。右手のパターンは上から(または下に)重なります(ポリメーター)。
