The dollar (

``` Haskell
$
```

) is a mysterious thing. It doesn't really do anything, but is super
useful. It's easy to get it mixed up with other operators in Tidal, for
example

``` Haskell
#
```

, because in a way they both 'join things together'. But what is

``` Haskell
$
```

, exactly?
