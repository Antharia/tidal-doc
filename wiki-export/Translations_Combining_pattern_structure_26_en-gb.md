This is very similar to how

``` Haskell
|+|
```

used to work in the versions of tidal prior to 1.0.0 - it took structure
from the left, but values from the right. In fact,

``` Haskell
#
```

is an alias for

``` Haskell
|>
```

, mirroring the behaviour in previous versions of tidal.
