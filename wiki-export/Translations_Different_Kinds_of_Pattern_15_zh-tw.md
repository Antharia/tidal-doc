若是你改用

``` Haskell
{ }
```

括號而非

``` Haskell
[ ]
```

括號，將有不同的結果。

``` Haskell
[ ]
```

每個子序列都在一個cycle時間內結束，產生複節奏(polyrhythm)的結構。

``` Haskell
{ }
```

則是以左手邊的pattern為主結構，右邊的pattern在主結構之上變化其值，產生混合拍的結構(polymetric
or polymeter)。
